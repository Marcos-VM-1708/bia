Questão3.v2: é um projeto pre desenvolvido onde decorri de uma solução e imprementei no codigo com pygame, que apresenta o jogo através de sprites utilizando o
conceito da matriz, utilizando os splites como índices para construção do jogo.
Resolvi adotar essa solução pois não tive tempo suficiente pra imprementar questão3
Nota-se que a pasta possui 3 arquivos sendo:
battle_ship_sprites = 4 tipos de sprite pro jogo
trata_sprite = separa e tradas os splite separados para o uso
questão_3_v.2 = codigo fonte


obs: codigo esta comentado funcionalidade por funcionalidade porém apresenta 2 princinpais bugs na
saida pro usuario(primeiro, é possivel alocar uma embarcação dentro da outra)(segundo, o jogador pode perder a vez se clicar
em um splite ja setado com elemento == 3)
